Hacer de una Raspberry una faasd

  • OpenFaaS

    OpenFaaS es un marco de código abierto que permite implementar la arquitectura sin servidor en Kubernetes usando contenedores de Docker para almacenar y ejecutar funciones. Permite que cualquier programa se encapsule como un contenedor y se administre como una función a través de la línea de comandos o la IU web integrada.

  • faasd

    Faasd ofrece mucho de los beneficios de los containers y OpenFaaS, pero sin la complejidad y los costos operacionales de Kubernetes. Se puede decir que es una versión más pequeña de OpenFaaS, diseñada para correr en un single host sin la necesidad de ningún clustering, por lo que es ideal para implementación de la tecnología function as a service dentro de una raspberry, debido a los bajos requerimientos computacionales.

  • Funciones

    Desde los botones a la izquierda de la pantalla se puede acceder a las funciones a las que se les hizo el deploy desde la raspberry, para obtener los valores de temperatura y humedad respectivamente.
